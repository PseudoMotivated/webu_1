Keep in mind for web development

For when you develop a website

  • Impactful, uncomplicated design
  • Coherent shapes and colors
  • Resiveness goes a long way
  • Intuitive navigation
  • Not too crammed
  • Have little media embedded, it's 2024.

For when a team will make a website

  1. Write a spec
  2. Make a shared repo
  3. Designate who will do what
  4. Make some tests to make sure the code works before you push to prod.
  5. If you roll your own crypto I will chase you in the dark.
  6. Make sure it works on mobile.
  7. Don't use email. Do DC or slack ffs.
